## Requirements



To use the GBPrimePay Checkout, you'll first need to [register for an GBPrimePay account](https://www.gbprimepay.com/).



== Installation ==



= Manual installation =



1. Download the GBPrimePay Checkout for WooCommerce .zip file of the latest release from here:
https://github.com/GBPrimepay/woocommerce-gbprimepay-checkout/releases

2. Upload the plugin files to the `/wp-content/plugins/` directory.

3. Activate the plugin through the "Plugins" screen in menu.

4. Use the GBPrimePay on Main Top Menu > GBPrimePay Settings screen to configure the plugin.
